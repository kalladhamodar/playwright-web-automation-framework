# Playwright Web Automation Framework (TypeScript)

## 📌 Overview
This project demonstrates a scalable automation framework using Playwright with TypeScript for web application testing.

This framework is inspired by real-time testing experience on subscription-based platforms with device-based access control.

---

## 🚀 Features
- End-to-end web automation using Playwright
- Automation scripts using TypeScript
- Page Object Model (POM) architecture
- Cross-browser testing support
- CI/CD ready structure (Jenkins compatible)

---

## 🌐 Test Coverage
- User login scenarios
- Subscription workflows
- One user – one device validation
- Cross-platform testing scenarios

---

## 🛠 Tech Stack
- Playwright
- TypeScript
- JavaScript
